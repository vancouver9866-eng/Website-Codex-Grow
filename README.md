# Growcean Next.js Homepage

Reference homepage built from the supplied website brief, product catalog, logo,
founder photo and banner.

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Notes

- The inquiry form is an interactive front-end prototype and does not transmit data.
- Unconfirmed certificates, factory scale, project cases and customer claims are intentionally excluded.
- Product availability and smart-control compatibility should be confirmed before publishing.
